# Smilyism16 - Interactive UI Pro Max Components

A comprehensive React UI component library with animations inspired by modern design systems like Figma UI Kits, Untitled UI, and Frames X.

## Features

- ✨ **Interactive Animations** - Built with Framer Motion
- 🎨 **Modern Design** - Claude AI inspired color palette
- 📱 **Responsive** - Mobile-first approach
- 🌓 **Dark/Light Modes** - Full theme support
- ⚡ **High Performance** - Optimized React components
- 🎯 **Accessibility** - WCAG compliant

## Components Included

- Buttons (Primary, Secondary, Icon, Loading states)
- Chat Bubbles (User & AI styles)
- Text Inputs & Forms
- Cards & Modals
- Loaders & Spinners
- Navigation (Sidebar, Top Nav)
- Avatars
- Badges & Tags
- Toast Notifications

## Getting Started

```bash
npm install
npm run dev
```

Visit `http://localhost:5173` to see the component showcase.

## Project Structure

```
src/
├── components/       # Reusable UI components
├── animations/       # Framer Motion configurations
├── styles/          # Tailwind CSS & theme files
├── pages/           # Component showcase pages
└── App.jsx          # Main application
```

## Tech Stack

- React 18
- Vite
- Tailwind CSS
- Framer Motion
- TypeScript (optional)
